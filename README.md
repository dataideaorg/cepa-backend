# CEPA Backend

## About

The CEPA (Centre for Policy Analysis) backend is a Django REST API that powers the CEPA website. CEPA is a leading think tank in Uganda dedicated to strengthening democratic governance, promoting transparency and accountability, and advancing human rights through rigorous policy research, parliamentary monitoring, and evidence-based advocacy.

## Features

- **Content Management**: RESTful API for managing website content including blog posts, news articles, publications, and events
- **Focus Areas**: Management system for policy focus areas with activities, resources, and partners
- **About Section**: Team members, partners, and organizational information management
- **Fellowships**: Cohort and fellowship program management
- **Get Involved**: Announcements, career opportunities, and contact form handling
- **Multimedia**: Gallery, videos, and podcasts management
- **Contact & Newsletter**: Contact form submissions and newsletter subscription management
- **Chatbot Integration**: AI-powered chatbot service for user interactions
- **Rich Text Editing**: CKEditor integration for content creation and editing
- **Admin Panel**: Django admin interface for content management

## Technology Stack

- **Framework**: Django 5.2
- **API**: Django REST Framework
- **Database**: PostgreSQL
- **Rich Text Editor**: CKEditor
- **AI Integration**: Anthropic (for chatbot)
- **Deployment**: Gunicorn, WhiteNoise for static files
